# Data Cleaning

A collection of Jupyter notebooks demonstrating data cleaning and preprocessing techniques on a variety of real-world and practice datasets. Each folder focuses on a different dataset and walks through the steps needed to take raw, messy data to an analysis-ready state.

## 📁 Repository Structure

| Folder | Description |
|---|---|
| `Data cleaning ( Amazon_products Dataset)` | Cleaning product listing data (prices, ratings, categories, etc.) |
| `Data cleaning ( E-commerce Dataset)` | Cleaning general e-commerce transaction/order data |
| `Data cleaning ( HR Analytics Dataset)` | Cleaning employee/HR records for analytics |
| `Data cleaning ( credit_risk_customers Dataset)` | Cleaning customer credit risk data |
| `Data cleaning (Netflix Dataset)` | Cleaning Netflix titles/catalog data |
| `Data cleaning (Online Food Dataset)` | Cleaning online food delivery/order data |
| `Data cleaning (Onurbltc SalesData)` | Cleaning sales transaction data |
| `Data cleaning (Practise Dataset)` | General practice notebook for data cleaning techniques |
| `Data cleaning (Students Exam Scores Extended Dataset)` | Cleaning student exam performance data |
| `Data cleaning (Zomato Dataset)` | Cleaning restaurant/food delivery data from Zomato |

Each folder typically contains:
- The raw dataset (CSV) used as input
- A Jupyter Notebook (`.ipynb`) with the data cleaning workflow
- (Optionally) a cleaned/output version of the dataset

## 🧹 What's Covered

Across these notebooks, common data cleaning tasks include:
- Handling missing values (imputation/removal)
- Removing duplicate records
- Fixing inconsistent data types
- Standardizing text/categorical values
- Detecting and treating outliers
- Renaming and reformatting columns
- Parsing and converting date/time fields

## 🛠️ Tech Stack

- **Python**
- **Pandas** for data manipulation
- **Jupyter Notebook** (run via **VS Code**)

## 🚀 Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/24a31a05fx/Data-cleaning.git
   cd Data-cleaning
   ```
2. Install the required dependencies
   ```bash
   pip install pandas jupyter
   ```
3. Open the folder in VS Code and run the notebook (`.ipynb`) you're interested in using the Jupyter extension.

## 📌 Purpose

This repository serves as a personal practice/portfolio collection for learning and demonstrating data cleaning and preprocessing skills across diverse datasets — a key step in any data analysis or machine learning pipeline.
